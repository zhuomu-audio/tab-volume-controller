Privacy Policy

Last updated: August 24, 2025

1. Purpose of the Extension

This extension allows users to independently control the volume of each browser tab. Its sole purpose is tab volume control. Users can adjust or mute tabs individually, and the extension remembers these settings for each tab.

2. Remote Code

This extension does not load or execute any remote code. All code and resources are included within the extension package. No external servers are contacted for functionality.

3. Host Permissions (<all_urls>)

The extension requires access to all URLs to identify active tabs and control their audio volume. No web page data is stored, transmitted, or shared.

4. Scripting Permission

The extension uses the Chrome Scripting API to inject scripts into tabs to control their audio volume. These scripts only manipulate audio settings and do not access any personal data.

5. Storage Permission

The extension uses Chrome Storage to persist user volume settings for each tab locally. This data is stored only on the userâ€™s device and is not transmitted externally.

6. Tabs Permission

The extension uses Chrome Tabs API to identify open tabs and apply volume control settings. No tab content, URLs, or browsing data is collected or transmitted.

7. Data Collection

This extension does not collect, transmit, or share any personal or sensitive user data. All settings are stored locally and used solely to enhance the user experience.
